# pattern printing

i --> row number
j --> column number

m --> number of column
n --> number of rows

cout << (* / j / i)  

- square    : i<=m , j<=m 

- rectangle : i<=m , j<=n 

- triangle  : i<=m , j<=i

- horizontal flipped triangle : 

  i<=m , j= (m-i+1)   
  OR
  i<=m , (j=m ,j>=i,j--)
